## Purpose
The purpose of this portfolio is to show my experience, skills and personality to potential employers, customers and other web developers. 

## Functionality/ Features
CSS and HTML were used in this webpage
Social media links are linked on the homepage
Home, About, Services, Blog and Contact are tabs featured in this portfolio which are all linked
A resume is linked to the Home page 
Images are linked in the Home, About, Services and Blog pages
A logo is always visible on the top left 
A HTML form is used in the contact page 
Each blog article is linked to another html file with text
